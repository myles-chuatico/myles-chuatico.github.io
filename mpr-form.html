Back-End Automation Layer
Because Freshservice does not natively transfer form inputs to ticket fields, a suite of workflow automators was built to handle this gap:

· Category & subcategory, set from MPR type
· BA code, routed from mission selection
· Finance unit, assigned based on agent group
· Agent group, auto-assigned on approval
· Threshold routing, >USD 3,000 escalates to Senior Agent

No manual data re-entry required despite manual form submission.
