Automation Project · HSBC Financial Support Dept

eLinq Utilization
Reporting Pipeline

A self-initiated automation project that transformed fragmented, manually-collected log data into a daily-refreshing centralized utilization dashboard, giving leadership the data to improve system adoption across the department.

Problem
AVPs flagged declining call volumes but had no data on eLinq adoption. Log files sat on individual agent machines with no collection process. Reports were manual, inconsistent, and days late.

🖥️
Agent Machines
eLinq log output
Each agent's machine generates daily eLinq activity logs as part of normal tool usage. 100+ agents across the department, each with a local log file.
Raw Logs
02 / COLLECT
📜
Batch Script
Windows .bat automation
Agent machines were locked-down corporate endpoints with no access to third-party automation tools or admin-level software. Windows Task Scheduler was the only available automation mechanism. A batch script was deployed to each machine, scheduled to run 10 minutes before end of shift, copying the day's eLinq log file to a mapped shared network folder.

Technical Detail
Batch Script Logic
Automated Log Collection
Agent machines were restricted corporate endpoints with no access to third-party tools or elevated privileges. Windows Task Scheduler was the only viable automation option. The batch script runs 10 minutes before end of shift, when the day's eLinq activity is complete. It copies the log to the shared network path and handles file naming (agent ID + date) to prevent overwrites and keep attribution clean.
:: Daily eLinq log push
@echo off
set DATE=%date:~-4%-%date:~4,2%
xcopy "%LOCALAPPDATA%\eLinq\*.log"
    "\\server\shared\elinq-logs\"
    /Y /Q
Power Query Steps
Data Cleaning Pipeline
Power Query M script connects to the shared folder, combines all files, and applies a standardized set of cleaning transformations that run automatically on each daily refresh.
// Power Query M steps
1. Folder.Files(shared_path)
2. Combine & expand all logs
3. Parse timestamp column
4. Remove null/error rows
5. Add agent_id from filename
6. Pivot by date + agent
